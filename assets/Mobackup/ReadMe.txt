                 MOBACKUP - Datensicherung für Outlook

         Software zum Sichern von E-Mail-Daten und Einstellungen in
      Outlook 2000, Outlook XP, Outlook 2003, Outlook 2007, Outlook 2010,
         Outlook 2013, Outlook 2016, Outlook 2019 und Outlook 2021
         
-----------------------------------------------------------------------------

HILFE
============

Eine ausführliche Anleitung zur Funktionalität der Software finden Sie in der
beigefügten Hilfe, die Sie innerhalb der Software jederzeit über die Taste F1
bzw. über die entsprechenden Hilfe-Schaltflächen aufrufen können.



DEINSTALLATION
==============
Zur Deinstallation der Software klicken Sie auf START - SYSTEMSTEUERUNG.
Dort starten Sie mit einem Doppelklick auf SOFTWARE (ab Windows Vista
PROGRAMME UND FUNKTIONEN) die Auflistung der zur Zeit installierten Programme.

Wählen Sie "MOBackup - Datensicherung für Outlook (Vollversion)" und
klicken Sie auf ÄNDERN/ENTFERNEN (ab Windows Vista DEINSTALLIEREN/ÄNDERN).

Folgen Sie den Anweisungen.
