PYTHON OFFLINE-INSTALLATION
============================

Dieser Ordner ist für die Offline-Installation von Python gedacht.

ANLEITUNG:
1. Laden Sie Python von https://www.python.org/downloads/ herunter
   Empfohlen: Python 3.11.9 (64-bit)
   Direktlink: https://www.python.org/ftp/python/3.11.9/python-3.11.9-amd64.exe

2. Speichern Sie die Datei in diesem Ordner mit dem Namen:
   python-3.11.9-amd64.exe

3. Wenn Python nicht installiert ist, verwendet start.bat automatisch
   diesen lokalen Installer statt einen Download zu versuchen.

VORTEIL:
- Funktioniert ohne Internetverbindung
- Schnellere Installation
- Ideal für USB-Stick oder Netzlaufwerk-Verteilung
